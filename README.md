# CareerQuest

## Project Setup

### Prerequisites
- Node.js
- MySQL

### Database
1. Create a MySQL database named `careerquest`.
2. Run the schema script: `mysql -u root -p careerquest < database/schema.sql`

### Backend
1. Navigate to `backend` folder.
2. Run `npm install`.
3. Create a `.env` file with your DB credentials.
4. Run `npm start` (or `node server.js`).

### Frontend
1. Navigate to `frontend` folder.
2. Run `npm install` (if not already done).
3. Run `npm run dev`.

## Features
- **Modern Dashboard**: Clean, responsive UI with deep navy and gold accents.
- **Career Roadmap**: Interactive vertical stepper.
- **Bounty Grid**: Listing of tuition and freelance opportunities.
- **Interview Simulator**: Mock UI for AI-driven interviews.
- **Skill Radar**: Visualization of skill gaps.
